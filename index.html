Bienvenido al Simulador de Tiro Parabólico

Este sitio web fue desarrollado para facilitar el aprendizaje del
movimiento parabólico mediante una simulación interactiva y visual.
Aquí podrás modificar variables como el ángulo y la velocidad inicial
para observar el comportamiento de la trayectoria del proyectil en
tiempo real.
🚀 Aprende física de una manera dinámica, práctica y sencilla.
